## Cadmus Fake Player

Cadmus Fake Player is an addon for Odyssey Claims.

This mod allows chunks loaded by Odyssey Claims to simulate the presence of a player, even when no real player is nearby or connected.

### Purpose

In Minecraft, many game mechanics require a nearby player to function correctly. This includes things like crop growth, certain mob behaviors, and some automated systems.

Cadmus Fake Player solves this limitation by making chunks loaded and claimed by Odyssey Claims behave as if a player were present. This allows game mechanics to continue functioning normally, even when no other players are online.
